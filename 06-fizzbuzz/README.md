# 06 - FizzBuzz

## Description

Print numbers from 1 to N.  
- For multiples of 3, print "Fizz"  
- For multiples of 5, print "Buzz"  
- For multiples of both 3 and 5, print "FizzBuzz"

## Input

- A single integer N

## Output

- Numbers from 1 to N following the FizzBuzz rules

## Example

Input:  
15  

Output:  
1  
2  
Fizz  
4  
Buzz  
Fizz  
7  
8  
Fizz  
Buzz  
11  
Fizz  
13  
14  
FizzBuzz  

## Notes

- N is a positive integer
