# 04 - Find Maximum

## Description

Write a program that finds the maximum value in an array of integers.

## Input

- An array of integers

## Output

- A single integer representing the largest value in the array

## Example

Input:  
[3, 7, 2, 9, 5]  

Output:  
9  

Input:  
[-1, -5, -3]  

Output:  
-1  

## Notes

- The array may contain negative numbers
