# 02 - Sum Array Elements

## Description

Write a program that calculates the sum of all elements in an array of integers.

## Input

- An array of integers

## Output

- A single integer representing the sum of all elements

## Example

Input:  
[1, 2, 3, 4, 5]  

Output:  
15  

Input:  
[-2, 5, 7]  

Output:  
10  

## Notes

- The array may contain negative numbers  
- If the array is empty, the sum should be 0
