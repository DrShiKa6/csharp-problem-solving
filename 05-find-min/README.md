# 05 - Find Minimum

## Description

Write a program that finds the minimum value in an array of integers.

## Input

- An array of integers

## Output

- A single integer representing the smallest value in the array

## Example

Input:  
[3, 7, 2, 9, 5]  

Output:  
2  

Input:  
[-1, -5, -3]  

Output:  
-5  

## Notes

- The array may contain negative numbers  
- Assume the array contains at least one element
