# 01 - Simple Calculator

## Description

Write a program that performs basic arithmetic operations.

Supported operations:
- Addition
- Subtraction
- Multiplication
- Division

## Input

- First number
- Operator (+, -, *, /)
- Second number

## Output

The result of the operation.

## Example

Input:
10 + 5

Output:
15

## Notes

- Handle division by zero.
- Validate the operator.
