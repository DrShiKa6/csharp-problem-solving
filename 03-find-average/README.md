# 03 - Find Average

## Description

Write a program that calculates the average of all elements in an array of integers.

## Input

- An array of integers

## Output

- A single number representing the average of all elements

## Example

Input:  
[2, 4, 6, 8]  

Output:  
5.0  

Input:  
[1, 3, 5]  

Output:  
3.0  

## Notes

- The array may contain negative numbers  
- If the array is empty, the average should be 0
